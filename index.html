自动解密脚本工具

GitHub 配置

使用说明：
1. 获取 GitHub Token：前往 GitHub Settings → Personal access tokens
2. Token 权限：勾选 repo 权限
3. 仓库地址格式：用户名/仓库名
4. 确保仓库已集成 decode_action 解密框架
